# java-lotto-precourse

## 기능 구현 목록
> 기능 작동 순서대로 작성

1. 구입금액 입력 및 검증
   - 예외 사항
     - 숫자가 아닌 경우 -> Validator
       - 잘못된 형식입니다.
     - 1000원 단위(0 초과)가 아닌 경우 -> Money
       - 1000원 단위로 입력해야 합니다.
2. 구입 금액 저장
3. 금액에 맞는 개수만큼 로또 발행
4. 발행한 로또 출력
   - 개수
   - 로또 번호
5. 당첨 번호 입력 및 검증
   - 예외 사항
     - 잘못된 형식인 경우 -> Validator
       - 잘못된 형식입니다.
     - 당첨 번호가 중복인 경우 -> WinningLotto
       - 번호가 중복입니다.
     - 1~45 정수가 아닌 경우 -> WinningLotto
       - 로또 번호는 1부터 45 사이의 숫자여야 합니다.
     - 로또 번호가 6개가 아닌 경우 -> WinningLotto
       - 로또 번호는 6개여야 합니다.
6. 보너스 번호 입력 및 검증
   - 예외 사항
     - 숫자가 아닌 경우 -> Validator
       - 잘못된 형식입니다.
     - 당첨 번호와 중복인 경우 -> WinningLotto
       - 보너스 번호가 당첨 번호와 중복입니다.
     - 1~45 정수가 아닌 경우 -> WinningLotto
       - 로또 번호는 1부터 45 사이의 숫자여야 합니다.
7. 당첨 통계 계산
   - 등수별 당첨 개수 반환 -> WinningLotto에서 Result로 반환
8. 당첨 통계 및 수익률 출력
   - 수익률은 Result 내 메서드로 계산
